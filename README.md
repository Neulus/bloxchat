# BloxChat

BloxChat is a small desktop chat app for ROBLOX built with:

- React + TypeScript (UI)
- Tauri (desktop wrapper)
- Rust (native backend used by Tauri)

## Install (Easy Way)

If you just want to use the app:

1. Open this repo on [GitHub](https://github.com/logixism/bloxchat).
2. Click the **Releases** tab.
3. Download the installer.
4. Run the installer and open **BloxChat**.

## Use

1. Launch **BloxChat** from your app menu/start menu.
2. Sign in or connect as prompted by the app.
3. Join any ROBLOX game and start chatting!

If a new version is available, download it from the GitHub **Releases** tab the same way.
